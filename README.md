# City Event Connect

City Event Connect is a community-driven event discovery platform
designed to connect multiple colleges within a city through a
centralized event management system.

This project addresses the communication gap between institutions by
providing a structured platform where students can explore and register
for academic, technical, and cultural events.

In many cities, colleges conduct workshops, hackathons, tech fests, and

However:

-   Students from other colleges are often unaware of these events.
-   Event information is scattered across social media and messaging
    groups.
-   There is no centralized academic event ecosystem.

This leads to missed opportunities and reduced participation.
